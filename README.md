♻️ Swappy – Plataforma de Troca e Venda Sustentável
O Swappy é um projeto acadêmico desenvolvido como Trabalho de Conclusão de Curso (TCC) do curso Técnico em Desenvolvimento de Sistemas da ETEC Prof. Horácio Augusto da Silveira.

A plataforma tem como objetivo incentivar a economia circular, promovendo a troca e venda de itens entre usuários, reduzindo o desperdício e estimulando práticas de consumo consciente e sustentável.

O sistema oferece um ambiente digital seguro, acessível e organizado, permitindo que usuários cadastrem itens, negociem trocas, interajam entre si e acompanhem suas transações de forma transparente.

🎯 Objetivo do Projeto
Desenvolver uma solução tecnológica que facilite a troca e venda sustentável de itens, promovendo:

Reaproveitamento de recursos
Redução de resíduos
Consumo consciente
Colaboração entre usuários

📱 Funcionalidades Principais

🧩 Arquitetura do Sistema
O projeto segue uma arquitetura baseada em:

Front-end Web e Mobile para interação com o usuário
Back-end em Java responsável pelas regras de negócio
API REST para comunicação entre cliente e servidor
Banco de dados MySQL para persistência das informações

✅ Status do Projeto
O projeto Swappy encontra-se finalizado, tendo sido desenvolvido e entregue como Trabalho de Conclusão de Curso (TCC) do curso Técnico em Desenvolvimento de Sistemas da ETEC Prof. Horácio Augusto da Silveira.

Todas as funcionalidades previstas no escopo acadêmico foram implementadas, documentadas e apresentadas à banca avaliadora.
Atualmente, o repositório tem caráter exclusivamente acadêmico e histórico, não estando em manutenção ativa nem recebendo novas atualizações de código.

O projeto permanece disponível para fins de estudo, consulta e demonstração técnica, representando a aplicação prática dos conhecimentos adquiridos ao longo da formação.
